# Demo Guide - 7 Brew Mobile App

## Quick Demo Flow (5 minutes)

### 1. Home Screen (30 seconds)
- Show personalized welcome with user's name
- Point out **current points (750)** and progress bar
- Highlight **1 free drink available**
- Show quick category browsing
- Demonstrate featured drinks

### 2. Browse Menu (1 minute)
- Navigate to Menu from bottom nav
- Show category filtering (Coffee, Energy, Smoothies, etc.)
- Demonstrate smooth scrolling
- Point out caffeine level badges
- Show signature and seasonal tags

### 3. Customize a Drink (1.5 minutes)
**Select "Blondie" (most popular drink)**
- Show size selection (Small/Medium/Large)
- Change temperature (Hot/Iced/Frozen)
- Adjust milk options (Oat milk is popular)
- Modify espresso shots
- Show sweetness levels
- Expand advanced options:
  - Add/remove syrups
  - Add toppings (cold foam, caramel drizzle)
- Add to cart with animation

### 4. Cart & Checkout (1.5 minutes)
- View cart with customization summary
- Show quantity adjustments
- Display order total with tax
- Proceed to checkout
- Select pickup location
- Choose ASAP or scheduled pickup
- **Show free drink toggle** (you have 1 available)
- Display demo payment card details:
  - Card: 4242 4242 4242 4242
  - Expiry: 12/28
  - CVV: 123
  - ZIP: 77304
- Place order
- Show success animation
- Mention **100 points earned per drink**

### 5. Rewards System (30 seconds)
- Navigate to Rewards tab
- Show **750/1000 points** progress
- Display "3 drinks until next reward"
- Show lifetime statistics:
  - 5,750 lifetime points
  - 2 locations visited
  - 12-day longest streak
  - 58 total visits
- Scroll through recent activity

### 6. Other Features (30 seconds)
- **Orders**: Show order history with reorder button
- **Profile**: Display account details and favorites
- **Locations**: Show 2 Conroe, TX locations with directions
- Mention 553 stands nationwide
